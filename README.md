A lightweight web app to help users track their active screen time, get mindful reminders, and build healthier digital habits through social detox and digital wellbeing.

---

## Features

- Automatic tracking of active screen time based on user activity (mouse, keyboard, touch)
- Manual start/stop session controls
- Daily usage summary and session history
- Periodic reminder notifications to encourage breaks
- Customizable settings for daily limit and reminder intervals
- 7-day usage visualization via bar chart
- Data persistence in browser localStorage (no backend needed)
